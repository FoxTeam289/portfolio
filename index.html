Проекты

Mushfind

Mushfind - это инновационный сервис, разработанный специально для грибников и любителей грибной охоты. С помощью Mushfind пользователи могут получить доступ к интерактивным картам, на которых отмечены лесные участки, и просмотреть отзывы о них, оставленные другими пользователями.

Прогноз погоды

Веб-приложение для прогноза погоды с использованием API, которое отображает актуальные данные о погоде в реальном времени.

Калькулятор

Интерактивный калькулятор с чистым и интуитивно понятным интерфейсом, который помогает выполнять основные математические операции.

Интеграция с ChatGPT

Внедрение ChatGPT API в веб-приложение для создания интерактивного и умного ассистента, который помогает пользователям в решении различных задач.
